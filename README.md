# GrantDAO

DAO-based grant distribution system.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/GrantDAO.sol`
- Stacks: `contracts/stacks/contracts/grantdao.clar`

## Setup
```bash
pnpm install
pnpm dev
```
